== README

Aplikacja biblioteki filmowej.

== Gems

gem 'turbolinks' (+ Aniamte.css) - płynne przejścia pomiedzy stronami,

gem 'carrierwave' - obsluga plików,

gem 'rmagick' - obróbka zdjęć,

gem 'flexslider' - wyświetlanie karuzeli z filmami,

gem 'fog' - połączenie bazy danych z plikami na Google Storage,
